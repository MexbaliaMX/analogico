# Recommended Educational Content

## Foundational Reading
- **“Analog Matrix Computing with Crosspoint Resistive Memory Arrays”** (Sun et al., IEEE TCAS-II, 2022) — accessible overview of in-memory analog computing principles and time-complexity arguments.
- **“Mixed-Precision In-Memory Computing”** (Le Gallo et al., Nature Electronics, 2018) — key paper on combining low-precision analog cores with digital refinement.
- **“Time Complexity of In-Memory Solution of Linear Systems”** (Sun et al., IEEE TED, 2020) — delves into convergence guarantees and analog solver stability.

## RRAM Device Physics & Fabrication
- **Coursera: Nanotechnology for Memory and Logic** (EPFL) — modules on resistive switching mechanisms and CMOS integration.
- **IEEE IEDM 2023 Tutorials on RRAM** — conference videos covering 1T1R stacks, endurance, and write–verify protocols.
- **“ASAP Programming Algorithm for Multi-Level RRAM Cells”** (Sun et al., IRPS 2024) — detailed treatment of the write–verify scheme used in HP-INV.

## Numerical Linear Algebra & Algorithms
- **MIT OpenCourseWare 18.335: Introduction to Numerical Methods** — lectures on iterative refinement, condition numbers, and matrix factorization.
- **“Iterative Refinement for Linear Systems”** (Higham, Acta Numerica) — theoretical backbone for hybrid analog–digital solvers.
- **Stanford CS237B: Linear Algebra for Machine Learning** — focuses on practical matrix inversion and conditioning, with assignments on MIMO detection.

## RF & Massive MIMO Applications
- **“An Overview of Massive MIMO”** (Lu et al., IEEE JSTSP, 2014) — foundational reading for RF engineers entering massive MIMO.
- **NPTEL: Wireless Communication Lab** — hands-on SDR experiments; pair with HP-INV demos for channel inversion practice.
- **Keysight RF University** — webinars on baseband processing bottlenecks and Gram-matrix inversion in 5G/6G systems.

## Tooling & Hands-On Labs
- **GitHub: Analog-In-Memory-Computing-Lab** — open-source notebooks simulating RRAM arrays with PySpice.
- **Analog Circuit Design with LTspice** (Texas Instruments training) — essential for prototyping op-amp feedback used in LP-INV.
- **CERN ROOT & Python for Data Acquisition** — reinforces signal processing and logging skills for hardware-in-the-loop validation.
